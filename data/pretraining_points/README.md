Pre-training points relevant to the AI4Snow project:

- `sampling_points_mountains_extended.csv` (by Sebastian Roessler): based on [GMBA dataset](https://www.gmba.unibe.ch), filtered by mountains with well-recognized names, mean snow cover duration >=50 days per year. This is the new version of the data (included extended part of the Rockies), retrieved on 25 November 2024, including 87 mountain ranges. (Note: South latitudes are indicated with a minus)
- `sampling_points_mountaints_lat_42-60.csv`: is based on the extended file, including only latitudes between 42-60 degrees North (len: 51213).
- `sampling_points_mountaints_remaining.csv`: contains the remaining data points (len: 98787).
- `galileo_global.csv` (by Gabriel Tseng): global (land-) sampling points, more documentation currently missing.
